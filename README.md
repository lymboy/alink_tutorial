# 项目简介

Alink 机器学习工程化解决方案教程

# 快速开始

> 直接使用本项目提供的相关依赖和版本就好，不然会有很多坑，非常麻烦，需要不断试错

+ **必须保证JDK == <span style="color:red;">1.8</span>**
+ 使用`maven`作为项目构建工具
+ 如果暂时不用`mysql`可以直接将相关`pom.xml`依赖注释
+ 推荐使用idea作为开发工具
